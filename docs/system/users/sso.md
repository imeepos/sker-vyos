# 系统管理 - 单点登录页面

## 页面功能

### 核心功能
- **SSO配置管理** - 配置和管理单点登录系统
- **身份提供商集成** - 集成各类身份提供商和认证服务
- **登录流程配置** - 配置SSO登录流程和策略
- **会话管理控制** - 管理用户会话和令牌生命周期

### 详细功能列表
1. SAML身份提供商配置
2. OAuth 2.0/OpenID Connect集成
3. LDAP/Active Directory集成
4. SSO登录策略配置
5. 多因子认证集成
6. 会话超时策略设置
7. 登录审计日志记录
8. SSO故障诊断工具
