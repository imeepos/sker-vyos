# 网络服务 - 5G/IoT安全防护页面

## 页面功能

### 核心功能
- **IoT安全威胁防护** - 防护IoT设备面临的各类安全威胁
- **设备安全认证** - 管理IoT设备的安全认证机制
- **数据传输加密** - 保护IoT数据传输的安全性
- **安全监控告警** - 监控IoT安全事件和异常行为

### 详细功能列表
1. IoT设备安全风险评估
2. 设备身份认证管理
3. 端到端数据加密配置
4. 安全证书生命周期管理
5. 异常行为检测分析
6. 安全事件响应处理
7. 安全策略合规检查
8. 安全态势感知展示
